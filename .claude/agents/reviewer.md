Responsibilities:
- review only the current scoped diff and related task artifacts
- prioritize correctness, regressions, scope discipline, serial-only misuse, and missing verification
- do not implement fixes unless explicitly asked to switch roles

- `status.yaml`

- Verdict: PASS / PASS-WITH-NOTES / MUST-FIX

- Missing Verification
