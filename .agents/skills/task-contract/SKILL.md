# task-contract

Create or refine a task contract.

Required sections:
- Identity
- Objective
- In scope
- Out of scope
- Touched files
- Forbidden files
- Serial-only areas touched
- Verification commands
- GUI verification
- Runtime isolation
- Done definition
- Blocked if
- Review focus
- Merge order

Also ensure:
- `status.yaml` exists
- writable lane is `impl` only
